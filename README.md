# 부진 재고 분류 기준 설계 및 분석
> **단순 판매량이 아닌 '판매 속도'와 '체류 기간'을 기준으로 부진 재고를 재정의한 프로젝트**

## 1. 프로젝트 개요 및 문제 정의
- **목적:** 상품군별(의류/신발) 특성에 맞춘 객관적인 부진 재고 판단 기준 설계
- **문제 인식:** 판매량 중심의 고정 임계값은 회전 속도와 체류 기간이 다른 상품군(의류는 체류에 민감, 신발은 긴 회전 허용)의 특성을 반영하지 못함
- **해결 방안:** 부진 재고를 절대 수치가 아닌, 동일 상품군 내 **'상대적 회전 지연 상품'**으로 재정의

---

## 2. 데이터 환경 및 분석 도구
- **분석 대상:** 의류(AP), 신발(FW) 전체 재고 (최근 3개월 누적 데이터)
- **비교 단위:** 카테고리 + 상품 형태 (`biz` + `silhouette`)
- **핵심 지표:** 판매 효율(`sale_rate`), 체류 기간(`days_since_ship`)
- **주요 기술:** Python, Pandas, Scikit-learn (MinMaxScaler, 데이터 정규화 및 가중치 최적화)

---

## 3. 핵심 분석 및 분류 프로세스

### 3-1. 탐색적 데이터 분석 (EDA)
- 재고량과 판매량은 양의 상관관계를 보이나, 회귀선에서 크게 벗어난 '재고 과다·판매 전멸' 상품 다수 확인.
- 장·단기 재고가 혼재되어 있어 '입고 후 경과일'이라는 시간적 축을 분석에 도입.

```py
# 기초 통계 및 결측치, 노이즈 확인
df.describe()
df[df["sale_qty"] < 0]

# 재고량과 판매량 상관관계 분석
df[["qty","sale_qty"]].corr()
```

### 3-2. 판단 기준 설계 및 스코어링
상대 스코어링 도입: 군집 분석이나 고정 임계값 대신, 동일 그룹 내 판매율 순위와 체류 기간 순위 산출.

MinMaxScaler 적용: 판매율(0~1)과 스케일을 맞추기 위해 체류 기간을 정규화하여 가중합 편향 최소화.

```py
from sklearn.preprocessing import MinMaxScaler

scaler = MinMaxScaler()

# 스케일링 진행
df["days_since_ship_scaled"] = scaler.fit_transform(df[["days_since_ship"]])
df["sale_rate_scaled"] = scaler.fit_transform(df[["sale_rate"]])

# 그룹 내 상대 순위 산출
df["sale_rank_in_group"] = df.groupby("category_type")["sale_rate"].rank(pct=True)
df["days_rank_in_group"] = df.groupby("category_type")["days_since_ship"].rank(pct=True)
```

### 3-3. 가중치 최적화 (Grid Search) 및 2축 분류
상품군별로 판매율과 체류일 가중치를 다르게 적용해 분류 구조를 보정.

Grid Search를 통해 최적의 가중치 도출 (신발은 판매율, 의류는 체류일에 더 높은 가중치 부여).

4가지 분류 유형:

    복합 부진: 판매 하위 & 체류 상위 (판매 저조 + 장기 체류)

    상품 부진: 판매 하위만 (신상품·반응 저조)

    체류 부진: 체류 상위만 (장기 재고)

    정상: 모두 미해당 (정상 회전)

```py
# 부진 유형 라벨링 로직
df["slow_label"] = "정상"

df.loc[(df["sale_rank_in_group"] <= 0.25) &
       (df["days_rank_in_group"] >= 0.60), "slow_label"] = "복합 부진"

df.loc[(df["sale_rank_in_group"] <= 0.25) &
       (df["slow_label"] == "정상"), "slow_label"] = "상품 부진"

df.loc[(df["days_rank_in_group"] >= 0.60) &
       (df["slow_label"] == "정상"), "slow_label"] = "체류 부진"
```

## 4. 인사이트 및 액션 플랜
### 핵심 인사이트
> 부진 재고는 단일 원인이 아니며, 특정 카테고리가 아닌 시점과 상품 특성에 따라 반복되는 패턴을 보임. 의류는 초기 판매율 저하 패턴이, 신발은 장기 체류 패턴이 뚜렷하게 확인됨.

### 부진 유형별 실행 방향

[복합 부진] 우선 조치 대상

    원인: 판매 저조 + 장기 체류 (자본 점유 심각)

    액션: 대형 할인, 묶음 판매, 조기 재고 정리

[상품 부진] 상품성 및 노출 점검

    원인: 초기 반응 저조 (상품성 또는 노출 문제)

    액션: 가격 정책 점검, 진열 방식 조정, 프로모션 테스트

[체류 부진] 구조적 회전 관리

    원인: 장기 보유 구조 문제

    액션: 시즌 전환 관리, SKU 축소, 발주 기준 재설정

### 확장 가능성
시즌/채널 데이터 결합
- 초기 판매 부진과 시즌 종료 재고를 구분
- 매장별 재고 분산 문제를 파악해 운영 구조 관점으로 관리 고도화 가능.
